## Getting Started

First, run the development server:

prepare .env file

```
# for twitter development
consumer_key=xxx
consumer_secret=xxx
access_token=xxx
access_token_secret=xxx

# for cotoha API
client_id=xxx
client_secret=xxx
```

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
